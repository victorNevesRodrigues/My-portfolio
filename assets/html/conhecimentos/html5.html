Estrutura Da Linguagem

Tags

O que são tags

Tags são marcadores que dizem para o navegador o que é aquele conteúdo que você digitou no código e como ele deve se comportar com ele.

Tipos de tag

Existem dois tipos de tags tags:
tags que tem tag de abertura e tag de fechamento.
tags sem conteúdo, essas abrem e fecham na mesma tag.

Diferença entre tag e elemento

Tags são usadas para marcar o início eo fim de um elemento Html
Já o elemento é o conjunto de tags + conteúdo

Atributos

O que são atributos

Atributos são palavras especiais usadas para mudar o comportamento das tags e dar informações extras sobre as mesmas.

Atributos globais e atributos locais

Existem atributos globais e atributos locais.
atributos globais podem ser usados em qualquer tag.
atributos locais são usados em apenas algumas tags.

Estrutura básica de uma pagina html

A estrutura básica de uma pagina html é formada por:

Instrução DOCTYPE, Tag html e dentro da tag html as tags:
head e body

Tag head

Head é uma tag container, dentro dela vão tags de configuração que em sua maioria não tem efeito visuais para o usuário. ex:
tag meta que serve para dizer por exemplo qual será o sistema de codificação dos caracteres, para dizer como os pixels devem se comportar em diversos dispositivos etc...
tag link que permite você importar outros tipos de arquivos, fontes externas etc...
tag title serve para dar nome um título ao documento esse título aparece na aba do navegador.

Tag body

Body é uma tag container, dentro dela vão as tags que serão visíveis para o usuário da aplicação.

comentários de código

comentários de código são muito úteis para dizer coisas que você quer que a pessoa que vai ler o seu código saiba, porém não quer que o usuário veja.

o propósito da linguagem

escopo da linguagem

HTML se delimita a criar a estrutura e a semântica do site, apesar de HTML ter algumas tags de estilização elas não devem ser usadas, estilização deve ser feita com o css.

Textos

Tags de criação de texto

Tags H

Tags H são tags de criação de cabeçalho, elas vão de h1 até h6, elas são divididas em ordem de importância
Sendo h1 a mais importante e h6 a menos importante, porém isso não tem nada ver com a maneira que elas são renderizadas pelo browser.

Tag p

A tag p é a tag utilizada para criação de parágrafos

Tags de formatação de texto

Só utilizo essas quando tem que ter significado semântico caso contrario foge do escopo do html, então eu uso css para fazer isso.

Tag b e tag strong

As tags b e strong são visualmente iguais porém elas tem uma diferença muito importante:
A tag b é somente uma forma visual de colocar o texto em negrito, então eu não uso ela e faço isso com o css
essa tag deixa o texto em negrito e semanticamente diz que o que estiver dentro dela é importante.

Tag i e tag em

As tags i e em são visualmente iguais porém elas tem uma diferença muito importante:
A tag i é somente uma forma visual de colocar o texto em itálico, então eu não uso ela e faço isso com o css
Já a tag em deixa o texto em itálico e semanticamente dá ênfase ao que estiver dentro dela.

Tag mark

A tag mark marca o texto como igual quando você passa um marca texto em um texto real.

Tag small e tag big

A tag small deixa o texto um pouco menor
A tag big deixa o texto um pouco maior.

Tag s e tag del

As tags s e del visualmente deixam o texto com um risco no meio dele, exemplo aqueles anúncios de x por y
a tag s só é visual mesmo
Já a tag del semanticamente diz que o texto dentro dela foi removido e substituído.

Tag u e tag ins

As tags u e ins visualmente colocam um risco em baixo do texto, o famoso underline
A tag u é apenas visual
Já a tag ins deixa o texto sublinhado e semanticamente diz que o texto foi inserido.

tag sub e sup

A tag sup deixa o texto dentro dela da metade para cima em relação ao texto padrão. ex: 2tm

A tag sub deixa o texto dentro dela da metade para baixo em relação ao texto padrão. ex: H2O

Tag blockquote, tag cite e tag q

A tag blockquote é usada para dizer semanticamente que aquele bloco de texto é uma citação, deve ser usada para citações mais longas, ela deve ser usado com o atributo cite, no atributo você vai colocar o link de onde tirou a citação
A tag cite semanticamente representa uma referência a um trabalho artístico exemplo o nome de um livro
A tag q é usada para citações curtas em linha, para citações maiores utilize blockquote.

Tag abbr

A tag abbr é a tag de abreviação utilizamos ela com o atributo title e o que colocarmos no title aparece quando a pessoa passar o mouse por cima da palavra abreviada.

Tag br e tag hr

A tag br é uma tag de quebra de linha, ele quebra não só texto mais também elementos, porém só deve ser usada em textos para quebra de elementos utilize o css
A tag hr cria uma linha horizontal, ela é usada para separar um tópico de outro.

Links

Forma de criar links

Tag a

Para criar links utilizamos a tag a + o atributo href esse atributo diz para onde o link aponta, o link pode apontar para 3 lugares:
Para uma página na web, colocando a url da página no href
para um arquivo na mesma pasta ou em uma pasta inferior ou superior a pasta atual, colocando o caminho dela no href
Para algum ponto da página exemplo é aqueles botões que mandam você para o topo da página, você cria um id para onde quer que o link aponte e depois referência ele no href ex: href"#topo"

Imagens

Como criar imagens

Tag img

A tag img serve para criar imagens, ela deve sempre ser acompanhada dos atributos src e alt
O atributo src diz da onde você está puxando a imagem, ele pode pegar imagens de 2 lugares:
Da web utilizando uma url no src
de um arquivo de imagem na mesma pasta ou em pasta superior ou inferior ao arquivo, colocando o caminho dela na src
O atributo alt cria um texto alternativo para caso a imagem não seja carregada e também diz para leitores de tela e mecanismos de busca do que se trata a imagem.

Tag figure e figcaption

As tags figure e figcaption são utilizadas quando você tem uma imagem e quer colocar uma legenda nela de forma semântica.
Você usa a tag figure de container dentro dela você coloca uma tag img com src e alt e abaixo da tag img coloca a tag figcaption o conteúdo da figcaption vai ser a legenda.

Imagens responsivas

Tag picture e tag source

As tags picture e source são utilizada para trocar as imagens que serão baixadas de acordo com o que você definir no atributo media, isso ajuda na eficiência e na responsividade da aplicação
A tag source deve ser usada com 2 atributos: media e srcset
o atributo media vai definir as condições a serem atendidas para a troca de imagens
O atributo srcset vai definir a imagem a ser usada quando a condição do media for atendida.
Você usa a tag picture como container e dentro dela coloca uma tag img + os atributos dela essa tag será a imagem padrão e abaixo dela define quantas tags source você quiser, lembrando que a cada tag source você deve colocar os atributos media e srcset nela.

Atributos de estilização de imagem

Existem atributos para fazer estilização de imagens, porém como não tem significado semântico eu não uso eles, porque fogem do escopo do HTML então eu uso o css para estilizar as imagens com o css.

Listas

Listas ordenadas

Tag ol e tag li

A tag ol define o inicio e o fim da lista e diz que é uma lista ordenada.
A tag li define os items da lista cada item deve ter um li

Listas não ordenadas

Tag ul e tag li

A tag ul define o inicio e o fim da lista e diz que é uma lista não ordenada.
A tag li define os items da lista cada item deve ter um li

Listas de descrição

Tag dl, tag dt e tag dd

A tag dl define o inicio e o fim da lista e diz que é uma lista de descrição.
dt define o termo de descrição. exemplo: café
dd define a descrição do termo. exemplo: é uma bebida preta, feita de grãos...

Tabelas

Tabelas semânticas

Tag table

A tag table delimita o inicio e o fim da tabela

Tag caption

No conteúdo da tag caption você coloca sobre o que é a tabela.

Tag thead

A tag thead delimita o inicio e o fim do cabeçalho da tabela, é uma tag semântica dentro dela vão as tags tr e th

Tag tr

A tag tr é a tag de criação de linhas da tabela, para cada linha deve ser usado um tr

Tag th

A tag th é a tag que cria as colunas de cabeçalho ou seja as categorias da tabela, para cada coluna use um th

Tag tbody

A tag tbody delimita o inicio e o fim do corpo da tabela, é uma tag semântica dentro dela vão as tags tr e td

Tag td

A tag td é a tag que cria as colunas do corpo da tabela, para cada coluna use um td

Tag tfoot

A tag tfoot delimita o inicio e o fim do rodapé da tabela, é uma tag semântica dentro dela vão as tags tr e th

Videos

Como colocar videos com HTML

Tag video

Utilizamos a tag video junto com os atributos src e controls para colocarmos vídeos na página.
Utilizando dessa forma colocamos apenas um vídeo.
Outra forma de colocar videos na página é utilizando a tag video + tag source com atributo src
Utilizando dessa forma podemos colocar vídeos reservas para caso o primeiro não for suportado pelo browser.

Atributos da tag video

a tag video tem diversos atributos, os mais importantes são:

controls

O atributo controls serve para habilitar os controles do vídeo, ela sempre deve ser colocada senão não será possível dar play no vídeo.

controlsList

O atributo controlsList com o valor nodownload desabilita a opção de baixar o vídeo.

autoplay

O atributo autoplay habilita o início automatico do vídeo, ela sempre deve ser usada com o atributo muted porque muitos browsers vão bloquear o autoplay se não usarmos o atributo muted.

muted

O atributo muted faz o vídeo iniciar mudo.

poster

O atributo poster cria uma capa para o vídeo ela aparece se o vídeo não tiver iniciado ou estiver pausado.

loop

O atributo loop faz o vídeo ficar reiniciando automaticamente.

Audios

Tag audio

Utilizamos a tag audio junto com os atributos src, type e controls para colocarmos vídeos na página.
Utilizando dessa forma colocamos apenas um audio.
Outra forma de colocar audios na página é utilizando a tag audio + tag source com os atributos src e type
Utilizando dessa forma podemos colocar audios reservas para caso o primeiro não for suportado pelo browser.

Atributo type

O atributo type serve para dizermos qual é o tipo de dado e o formato dele. exemplo: audio/mpeg

Atributos tag audio

A tag audio tem praticamente os mesmos atributos da tag video então não vou sitar eles novamente

Sites dentro de outro site

Como colocar sites dentro do site

Tag iframe

A tag iframe nos permite colocar sites ou trechos de sites dentro da nossa aplicação, ela deve ser usada com o atributo src.
Podemos utilizar conteúdos de diversos sites como o youtube e google maps, porém tem sites que bloqueiam isso como por exemplo o site do google
Assim como a tag vídeo a tag iframe tem diversos atributos de configuração, então eu prefiro olhar na documentação quando eu preciso de algum atributo do iframe.
