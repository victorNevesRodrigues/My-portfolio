O propósito da linguagem

Escopo da linguagem

O proposito das CSS é estilizar os elementos html e como eles são apresentados visualmente, e o ideal é que elas se delimitem a fazer apenas isso

Estrutura Das CSS

Sintaxe das CSS

A sintaxe das css são basicamente formadas de:
Seletor {Propriedade: Valor;}

Seletores

O que são seletores

Os seletores no css servem para selecionar os elementos criados com o html.
Existem diversos seletores os principais são:

Seletor através do nome da tag

Selecionando através do nome da tag o estilo que você definir será aplicado para todas as tags com aquele nome

Seletor de id

Utilizando #nomeDoID selecionamos o elemento pelo id

Seletor de class

Utilizando .nomeDaClasse selecionamos os elementos pela classe e os estilos definidos vão ser aplicados a todos os items que tiverem esse classe

Seletor global

utilzando o seletor * os estilos definidos irão ser aplicados a todos os elementos

Seletor de atributos

utilizando [nomeDoAtributo] podemos selecionar todos os items com aquele atributo

Selecionar tags aninhadas

podemos selecionar tags que estão dentro de outras utilizando o espaço, exemplo: ol li {color:blue;}
A cor azul será aplicada a todas as tags li, que estiverem dentro de uma tag ol

Usar diversos tipos de seletores ao mesmo tempo

Podemos usar varios seletores ao mesmo tempo utilizando a virgula como separador, e com isso o estilo definido será aplicado a todos os seletores, fazendo isso podemos economizar linhas de código e deixar o código mais legivel

Propriedades

O que são propriedades

Propriedades são caracteristicas que você pode atribuir a um elemento html por exemplo a cor, elas são formadas por propriedade e valor
A propriedade diz qual a caracteristica que você quer mudar e o valor diz qual o valor daquela caracteristica que você quer mudar exemplo:
caracteristica = cor | valor = verde

Herança

O que é herança

Herança é o conceito das css que diz que quando a propriedade não for especificada para o elemento ela vai herdar esse propriedade do elemento pai, porém não são todas as propriedades que são herdadas do pai

Especificidade

O que é especificidade

Especificidade é o conceito que diz como o navegador vai decidir qual propriedade é mais relevante no caso de haver um conflito de propriedades, ela funciona de acordo com os seletores e quanto mais especificos os seletores forem mais relevantes eles se tornam,exemplo:
Se eu selecionar o elemento pelo nome da tag e definir a cor dele como rosa e selecionar o mesmo elemento pelo id dele e definir a cor como azul, o elemento ficará com a cor azul pelo fato de o id ser mais especifico que o nome da tag

Niveis de especificidade

Os niveis de especificidade são calculados da seguinte forma: css inline = 1000, ID = 100, Class = 10, Nome da Tag = 1, quando houver conflito entre os seletores o com maior pontuação será o escolhido

Cascata

O que é cascata

Cascata é o conceito que diz quando houver conflito em uma propriedade de um elemento e os seletores desse elemento tiverem o mesmo peso vai valer o elemento que veio por ultimo no documento ou seja mais abaixo no documento.

Box Model

O que é box model

Box model é um termo utilizado para falar sobre o design e layout, os elementos HTML são como se fossem caixas e essas caixas são formadas por conteúdo(content), espaçamento interno(padding), borda(border), espaçamento externo(margin)

Box Sizing

Por padrão quando você define a altura e largura de um elemento utilizando as propriedades width e height elas não incluem a borda e o espaçamento interno o que acaba causando alguns problemas ao estilizar o elemento
Box sizing veio para resolver isso, utilizando ele com o valor border-box o width e height que você definir vão incluir a borda e o pading no calculo, ou seja o valor que definir no height e width será o tamanho real que o elemento vai ter

Padding-top que colocará uma padding apenas no topo ex:
Padding-right que colocará padding apenas na direita ex:

Border

Border coloca uma borda que envolve o conteúdo e o padding.
border-top acrescenta uma borda ao top(os parâmetros de width, style e color devem ser colocados)
border-right acrescenta uma borda a direita(os parâmetros de width, style e color devem ser colocados)
border-bottom acrescenta uma borda ao bottom(os parâmetros de width, style e color devem ser colocados)
border-left acrescenta uma borda a esquerda(os parâmetros de width, style e color devem ser colocados)
border é um atalho para as propriedades acima e com essa propriedade podemos definir o tamanho, o tipo e a cor das 4 bordas ao mesmo tempo

Border-radius

Define o arredondamento das bordas, podemos definir de todas as bordas ou dos lados especificos que queremos

Cores no CSS

Propriedades Para Manipular As Cores

Background

Background é uma propriedade atalho que nos permite definir varias coisas do fundo do elemento, como por exemplo cores, tamanho, posição e etc...

Background-color

Essa propriedade define a cor do background

Color

Essa propriedade define a cor do conteúdo(texto)

Sistemas De valores de Cores Das CSS

Nome da cor

Podemos definir as cores atráves dos nomes delas em inglês, não uso muito porque o tom das cores pode variar de acordo com o navegador e tem apenas 140 cores, o que torna isso muito limitado

RGB

RGB forma as cores utilizando o cunjunto de cores vermelho, verde, azul com cada uma delas podendo ir de 0 a 255

RGBA

Funciona igual o RGB porém com o adendo do alfa o que nos permite controlar a opacidade da cor

Hexadecimal

Assim como o RGBA o hexadecimal é uma combinação de vermelho, verde, azul e alfa, porém cada uma das cores vai de 0 até f

Unidades De Medida

Fixas

São recomendadas para impressão porém não são muito boas para telas, então eu praticamente não uso elas com excessão do pixel

PX(Pixel)

O tamanho do px varia de acordo com a densidade da tela, basicamente é a menor elemento de uma tela

CM(Centimetros)

Podemos definir os tamanhos dos elementos em centimetros

MM(milimetros)

Podemos definir os tamanhos dos elementos em milimetros

IN(Polegadas)

Podemos definir os tamanhos dos elementos em polegadas

PT(Pontos)

PT varia de acordo com a densidade da tela podendo ir de 1px até varios px

PC(Picas)

1 PC é equivalente a 12 PT

Relativas

São recomendadas para impressão porém não são muito boas para telas, então eu praticamente não uso elas com excessão do pixel

EM

Essa unidade é igual ao tamanho da fonte no elemento que ela está sendo colocada

REM

Rem é igual o em, porém ela pega o tamanho da fonte da tag raiz ao invés da tag atual

VW

Vw é igual a 1% da largura da janela do navegador

VH

Vh é igual a 1% da altura da janela do navegador

VMIN

Vmin é igual a 1% do menor elemento da janela do navegador(altura/largura), ou seja se a altura for o menor o vmin utilizará o valor dela

VMAX

Vmax é igual a 1% do maior elemento da janela do navegador(altura/largura), ou seja se a altura for maior que a largura o vmax utilizará o valor dela

%

% é relativo ao tamanho do elemento pai

Fontes

Propriedades Para Manipular fontes

Font-family

Font-family altera a fonte padrão do elemento para a que você escolher.

Font-style

Define o estilo da fonte, essa propriedade tem dois valores: normal, italic.
Normal a fonte fica reta
Italic a fonte fica inclinada

Font-size

Controla o tamanho da fonte.

Font-weight

Controla o "peso" da fonte, da para deixar ela mais grossa(bold) ou mais fina.

Propriedades Para alterar textos

Word-wrap

Essa propriedade faz com que o texto seja quebrado para próxima linha se ele ultrapassar o tamanho do container.

Text-Align

Text-align define o alinhamento do texto
Essa propriedade tem 4 valores:
left - que alinha o texto a esquerda
right - que alinha o texto a direita
center - que centraliza o texto
justify - que faz o texto preencher todo espaço da direita e esquerda

Text-transform

Text-transform como o nome diz tranforma o texto.
essa propriedade tem 3 valores:
uppercase - transforma todo o texto em letra maiúscula.
lowercase - transforma todo o texto em letra minúscula.
capitalize - transforma a primeira letra de cada palavra em letra maiúscula.

Text-indent

Essa propriedade faz uma indentação no começo de cada parágrafo do texto ou seja da um espaço do tamanho que você definiu.

Text-decoration

Essa propriedade tem 4 parâmetros:
Underline - coloca uma linha abaixo do texto
Overline - coloca uma linha em acima do texto
Line-through - coloca uma linha no meio do texto
none - retira qualquer text-decoration que houver

Letter-spacing

como o nome sugere essa propriedade vai definir o espaço entre as letras.

Word-spacing

Essa propriedade define o espaço entre uma palavra e outra

Line-height

Como o nome sugere essa propriedade define a altura da linha ou seja o quanto uma vai ficar espaçada da outra em altura.

Text-shadow

cria uma sombra no texto, nela você vai colocar 4 parâmetros:
primeiro será o quão a direita do texto a sombra irá ficar
segundo o quão abaixo do texto a sombra irá ficar
terceiro o nível de desfoque da sombra(efeito blur)
quarta é a cor da sombra

Display

Essa propriedade define como um elemento é exibido na tela

none

O elemento irá sumir, como se nunca tivesse sido setado, porém ainda estará no código fonte

Block

O elemento vai ocupar todo o espaço horizontal da linha, width e height podem ser mudados

inline

Os elementos com display inline são renderizados em uma única linha, ocupando apenas o espaço necessário para seu conteúdo.
width e height não podem ser mudados e eles vão ser de acordo com o tamanho do conteúdo

inline-block

Igual o inline, porém o width e height podem ser alterados

Imagens como Background

Podemos utilizar imagens como background de elementos ao invés de cores.

Background

Background é uma shorthand property que nos permite definir varias coisas do background em uma só propriedade ex:
Definir a cor de fundo do background que pode ser uma cor ou um gradiente, definir imagem como background, posição, tamanho e etc...

Background-image

Essa propriedade define uma imagem como background do elemento

Background-repeat

Essa propriedade define se a imagem vai ser repetida caso ela seja menor que o container, também podemos definir o eixo que queremos repetir ou não repetir

Background-size

Essa propriedade define o tamanho da imagem de background

Background-position

Essa propriedade define a posição inicial da imagem

Background-attachment

Essa propriedade define se a imagem de background vai rolar junto com o resto da pagina ou se ela ficará fixa, isso é muito utilizado para fazer o efeito parallax

Layout

Float

Antigamente bastante usado para controle do layout, porém com o surgimento do flexbox e do grid se tornou bem menos relevânte na maioria dos casos

Z-index

Z-index define em qual camada o elemento irá ficar

Position

Position define o tipo de metodo de posicionamento que o elemento vai ter, e alguns valores fazem o elemento ir para outra camada, os 5 principais valores são:

static

Elemento padrão, o elemento vai ser renderizado como eles aparecem no documento

absolute

O elemento vai ser posicionado relativo ao elemento pai, caso este não seja static e assim por diante até chegar no body

relative

O elemento é relativo a posição inicial dele

fixed

O elemento é posicionado de acordo com a janela do navegador, e ele ficará sempre fixo naquele lugar mesmo que você rolar a pagina

sticky

No sticky o elemento é posicionado de acordo com a posição do scroll do usuário

Pseudo elementos

Pseudo elementos servem para estilizar uma parte de um elemento no total são 6, porém os 2 que são relevantes são:

::after

Usando after em conjunto com a propriedade content podemos inserir algum conteudo após o elemento

::before

Usando before em conjunto com a propriedade content podemos inserir algum conteudo antes do elemento

Pseudo classes

Pseudo classes são seletores no CSS que permitem aplicar estilos a elementos com base em eventos, estados específicos ou características.

:active

Essa pseudo classe é ativada durante momento que o usuário clicar em algo e manter o botão pressionado, mais utilizado em links

:checked

Pseudo-classe usada para selecionar elementos de formulário que estão marcados, como caixas de seleção ou botões de rádio.
