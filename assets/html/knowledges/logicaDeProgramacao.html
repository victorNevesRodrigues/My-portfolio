Algoritmo

O que é um algoritmo?

Um algoritmo é uma sequência finita e lógica de passos para realizar alguma tarefa ou resolver algum problema

Como construir um algoritmo

Podemos contruir um algoritmo utilizando 4 passos que são:
1. Entender o objetivo do algoritmo.
2. Definir as informações de entrada.
3. Fazer o processamento dos dados para chegar ao objetivo do algoritmo.
4. Definir as informações de saída do algoritmo

Formas de representação de um algoritmo

Um algoritmo pode ser representado de 3 formas que são:
1. Através de pseudocódigo.
2. Através de fluxogramas.
3. Atráves de uma linguagem natural

Como implementar algoritmos

Podemos implementar algoritmos atráves das linguagens de programação

Lógica de programação

O que é lógica de programação?

lógica de programação é a estruturação de conceitos e regras que orientam a execução de algoritmos

Variaveis

Variaveis são espaços na memória ram do computador que nos permitem armazenar ler e alterar dados conforme o necessario, são formados por:
Identificador, tipo, valor e endereço na memória

Identificador

O Identificador(nome) da variavel nos permite declarar, alterar e recuperar os valores das Variaveis

Tipo

Se a linguagem for de tipagem forte o tipo deve ser declarado ao criar a variavel e não pode receber dados que não sejam do tipo declarado
Se a linguagem for de tipagem dinâmica o tipo da variavel não precisa ser declarado e pode mudar de acordo com o valor atribuido a ela

Valores

Variaveis podem possuir valores de 2 tipos: Valores primitivos e valores passados por referência
Os primitivos são em geral: Number(dependendo da linguagem é dividido em inteiro e real), boolean, string e bytes, dependendo da linguagem existem mais tipos como null, undefined, BigInt e symbol.
Os passados por referência são: objetos, arrays, funções e dependendo da linguagem as strings.

Endereço na memória

Quanto mais baixo o nivel da linguagem mais é necessario é a manipulação do espaço em memória

Operadores

Operadores aritmeticos


Operadores unários

Os Operadores unários trabalham em um operando e eles são:

Unário Mais(+)

Unário Mais converte um operando em um número.

Unário Menos(+)

Unário Menos converte um operando em um número e torna o valor após ele negativo.

diferença entre prefix e postfix

Os operadores de incremento/decremento prefixo mudam o valor da variável antes da avaliação da expressão.
enquanto os operadores de incremento/decremento postfix mudam o valor da variável após a avaliação da expressão.

Incremento prefix e postfix

Como o nome indica ele incrementa o operando em 1

Decremento prefix e postfix

Como o nome indica ele decrementa o operando em 1


Operadores binarios

Por serem aritméticos eles seguem as regras de precedência da matemática.

Os Operadores binarios trabalham com um operador e dois operandos e eles são:

Operador de soma(+)

O operador soma funciona de duas formas:
Quando se refere a operandos do tipo numérico ele soma os dois operandos.
Quando se refere a strings ele concatena

Operador de subtração(-)

Subtrai o segundo operando do primeiro

Operador de multiplicação(*)

Multiplica os dois operandos

Operador de divisão(/)

Divide o primeiro operando pelo segundo

Operador de resto(%)

Divide o primeiro operando pelo segundo operando em uma divisão sem decimais e retorna o valor restante ex: 5/2 o modulo retornado é 1

Operador de potenciação(**)

Eleva o primeiro operando à potência do segundo operando

Operadores Relacionais

Os Operadores Relacionais servem para fazer a comparação entre dois valores e retornar um resultado booleano(verdadeiro ou falso).
Os 8 operadores são:

Menor que (<)

menor que serve para dizer que o primeiro valor é menor que o segundo e caso seja ele retorna true.
ex: 1<2 retorna true

Maior que (>)

Maior que serve para dizer que o primeiro valor é maior que o segundo e caso seja ele retorna true.
ex: 1>1 retorna false

Igual (==)

Serve para dizer que dois itens têm o mesmo conteúdo, porém ela não verifica se os tipos são iguais também.
ex: 8=="8" retorna true

Diferente (!=)

Serve para dizer que dois itens têm conteúdos diferentes e assim como o == não leva em conta os tipos dos itens.
ex: 8!="8" retorna false

Menor ou igual (<=)

Serve para comparar dois números e se o primeiro for menor ou igual ao segundo ele retorna true.
ex: 8<=10 retorna true

Maior ou igual (>=)

Serve para comparar dois números e se o primeiro for maior ou igual ao segundo ele retorna true.
ex: 8>=10 retorna false

Estritamente igual (===)

Serve para comparar a igualdade de dois itens tanto em conteúdo como em tipo.
ex: 8=="8" retorna false

Estritamente diferente (!==)

Serve para comparar se dois itens são diferentes tanto em conteúdo como em tipo.
ex: 8=="8" retorna true

Operadores De Atribuição

Os operadores de atribuição servem para atribuir ou fazer alguma operação com o valor da variável.
Os 6 operadores de atribuição são:

Atribuição (=)

È utilizada para você atribuir ou substituir o valor de uma variável.

Atribuição aditiva (+=)

È utilizada para acrescentar um valor a uma varíavel já existente ou para concatenar uma string com outra.

Atribuição subtrativa (-=)

È utilizada para subtrair um valor de uma varíavel já existente.

Atribuição multiplicativa (*=)

È utilizada para multiplicar o valor de uma varíavel já existente

Atribuição divisiva (/=)

È utilizada para dividir o valor de uma variável

Atribuição modular (%=)

È utilizada para dividir o valor da variável de forma inteira e obter o resto

Operadores Lógicos

Os Operadores Lógicos nos permitem fazer operações utilizando a lógica booleana e com isso retornar o resultado desejado.
Os 4 operadores lógicos são:

And (&&)

Para ele retornar verdadeiro tanto um operando quanto o outro tem que ser verdadeiros ou seja (v && v = true).
Caso um deles seja falso retorna false

or (||)

Para ele retornar verdadeiro se qualquer um dos operandos for verdadeiro ele retorna verdadeiro ou seja (v || v = true) (v || f = true) (f || v = true)

xor (!==)

Para ele retornar verdadeiro apenas um dos dois deve ser verdadeiro ou seja (v !== f = true) e (f !== v = true)

not (!)

Ele é utilizado para negar o valor de uma expressão ou seja ele inverte o valor dela (!v = false) (!f = true).

Operador Ternário

O operador condicional ternário é formado por uma condição que caso for verdadeira retorna um bloco de código e caso seja false retorna outro bloco de código.
Ele é uma alternativa boa ao if em certos casos.

Estruturas de Controle de fluxo

Estruturas Condicionais

Estruturas condicionais servem para dizer quando uma sentença ou bloco de código devem ser ou não executados.
As 3 estruturas condicionais são:

IF - Estrutura Condicional Simples

If se utiliza de uma expressão lógica que caso o resultado dela seja true ele executa o bloco de código definido dentro do if e caso seja false ela não executa

IF/ELSE - Estrutura Condicional Composta

No if else caso a expressão lógica seja true ele executa o bloco de código dentro do if e caso seja false ele executa o bloco de código dentro do else.
Caso seja necessario mais de duas condições podemos adicionar um if após o else e assim continuar a adicionar mais condições

Switch

Switch avalia uma expressão lógica e combina o valor da expressão com um case e caso eles sejam iguais executa a sentença de código atribuida aquele case, switch tem algumas particularidades porém não vou comentar sobre elas aqui

Estruturas de Repetição

Como nome diz essas estruturas servem para repetir uma sentença ou bloco de código um determinado ou indeterminado número de vezes..
As 3 estruturas de repetção são:

For

For foi criado para ser utilizado com quantidades determinadas de repetições e por isso é mais eficiente nesse tipo de situação

While

While foi criado para ser utilizado quando você tem uma quantidade indeterminada de repetições e é mais eficiente para isso

Do/While

Do while é parecido com o while a diferença é que o while checa a expressão primeiro e se for true executa o código
Já o do while executa o código uma vez e depois checa a condição

Vetores e Matrizes

Vetor

Um vetor é uma variavel que armazena uma coleção de dados de modo indexado e unidimensional.
Na definição formal da linguagem de programação os vetores são homogêneos(do mesmo tipo de dados) e de tamanho fixo, porém certas linguagens não seguem essa norma como no javascript por exemplo

Matriz

Matrizes são bem parecidas com vetores a diferença deles é que eles são bidimensionais ao invés de unidimensionais, ou seja além da dimensão das linhas elas também tem a dimensão das colunas
