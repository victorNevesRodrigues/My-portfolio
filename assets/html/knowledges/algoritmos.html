Conceito de algoritmo

O que é um algoritmo

Um algortmo é uma sequência finita e lógica de passos para realizar alguma tarefa ou resolver algum problema

Estrutura Das CSS

Sintaxe das CSS

A sintaxe das css são basicamente formadas de:
Seletor {Propriedade: Valor;}

Seletores

O que são seletores

Os seletores no css servem para selecionar os elementos criados com o html.
Existem diversos seletores os principais são:

Seletor através do nome da tag

Selecionando através do nome da tag o estilo que você definir será aplicado para todas as tags com aquele nome

Seletor de id

Utilizando #nomeDoID selecionamos o elemento pelo id

Seletor de class

Utilizando .nomeDaClasse selecionamos os elementos pela classe e os estilos definidos vão ser aplicados a todos os items que tiverem esse classe

Seletor global

utilzando o seletor * os estilos definidos irão ser aplicados a todos os elementos

Seletor de atributos

utilizando [nomeDoAtributo] podemos selecionar todos os items com aquele atributo

Selecionar tags aninhadas

podemos selecionar tags que estão dentro de outras utilizando o espaço, exemplo: ol li {color:blue;}
A cor azul será aplicada a todas as tags li, que estiverem dentro de uma tag ol

Usar diversos tipos de seletores ao mesmo tempo

Podemos usar varios seletores ao mesmo tempo utilizando a virgula como separador, e com isso o estilo definido será aplicado a todos os seletores, fazendo isso podemos economizar linhas de código e deixar o código mais legivel

Propriedades

O que são propriedades

Propriedades são caracteristicas que você pode atribuir a um elemento html por exemplo a cor, elas são formadas por propriedade e valor
A propriedade diz qual a caracteristica que você quer mudar e o valor diz qual o valor daquela caracteristica que você quer mudar exemplo:
caracteristica = cor | valor = verde

Herança

O que é herança

Herança é o conceito das css que diz que quando a propriedade não for especificada para o elemento ela vai herdar esse propriedade do elemento pai, porém não são todas as propriedades que são herdadas do pai

Especificidade

O que é especificidade

Especificidade é o conceito que diz como o navegador vai decidir qual propriedade é mais relevante no caso de haver um conflito de propriedades, ela funciona de acordo com os seletores e quanto mais especificos os seletores forem mais relevantes eles se tornam,exemplo:
Se eu selecionar o elemento pelo nome da tag e definir a cor dele como rosa e selecionar o mesmo elemento pelo id dele e definir a cor como azul, o elemento ficará com a cor azul pelo fato de o id ser mais especifico que o nome da tag

Niveis de especificidade

Os niveis de especificidade são calculados da seguinte forma: css inline = 1000, ID = 100, Class = 10, Nome da Tag = 1, quando houver conflito entre os seletores o com maior pontuação será o escolhido

Cascata

O que é cascata

Cascata é o conceito que diz quando houver conflito em uma propriedade de um elemento e os seletores desse elemento tiverem o mesmo peso vai valer o elemento que veio por ultimo no documento ou seja mais abaixo no documento.

Box Model

O que é box model

Box model é um termo utilizado para falar sobre o design e layout, os elementos HTML são como se fossem caixas e essas caixas são formadas por conteúdo(content), espaçamento interno(padding), borda(border), espaçamento externo(margin)

Box Sizing

Por padrão quando você define a altura e largura de um elemento utilizando as propriedades width e height elas não incluem a borda e o espaçamento interno o que acaba causando alguns problemas ao estilizar o elemento
Box sizing veio para resolver isso, utilizando ele com o valor border-box o width e height que você definir vão incluir a borda e o pading no calculo, ou seja o valor que definir no height e width será o tamanho real que o elemento vai ter

Cores no CSS

Propriedades Para Manipular As Cores

Background

Background é uma propriedade atalho que nos permite definir varias coisas do fundo do elemento, como por exemplo cores, tamanho, posição e etc...

Background-color

Essa propriedade define a cor do background

Color

Essa propriedade define a cor do conteúdo(texto)

Sistemas De valores de Cores Das CSS

Nome da cor

Podemos definir as cores atráves dos nomes delas em inglês, não uso muito porque o tom das cores pode variar de acordo com o navegador e tem apenas 140 cores, o que torna isso muito limitado

RGB

RGB forma as cores utilizando o cunjunto de cores vermelho, verde, azul com cada uma delas podendo ir de 0 a 255

RGBA

Funciona igual o RGB porém com o adendo do alfa o que nos permite controlar a opacidade da cor

Hexadecimal

Assim como o RGBA o hexadecimal é uma combinação de vermelho, verde, azul e alfa, porém cada uma das cores vai de 0 até f

Unidades De Medida

Fixas

São recomendadas para impressão porém não são muito boas para telas, então eu praticamente não uso elas com excessão do pixel

PX(Pixel)

O tamanho do px varia de acordo com a densidade da tela, basicamente é a menor elemento de uma tela

CM(Centimetros)

Podemos definir os tamanhos dos elementos em centimetros

MM(milimetros)

Podemos definir os tamanhos dos elementos em milimetros

IN(Polegadas)

Podemos definir os tamanhos dos elementos em polegadas

PT(Pontos)

PT varia de acordo com a densidade da tela podendo ir de 1px até varios px

PC(Picas)

1 PC é equivalente a 12 PT
