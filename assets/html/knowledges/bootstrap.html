O que é o Bootstrap

Bootstrap é uma biblioteca orientada a componentes.

Qual o objetivo do bootstrap

O objetivo do Bootstrap é fornecer uma biblioteca de componentes e estilos pré definidos o que deixa muito mais rapido e facil criar aplicações web responsivas

Como inserir o Bootstrap no projeto

Bootstrap tem 3 formas mais usadas de ser inserido no projeto que são:

Utilizando um CDN

CDN é um servidor externo que armazena os arquivos extáticos em cash, geralmente existem vários espalhados pelo mundo o que torna mais rapido para o navagedor do usuario baixar os arquivos e menos custoso retornar para o nosso servidor já que o arquivo é baixado de outro servidor

Utilizando arquivos internos

Podemos baixar Os arquivos do Bootstrap e colocarmos em um diretorio do nosso projeto e depois fazemos o link deles utilizando as tag link e script

Utilizando gerenciadores de pacotes

Eu utilizo o npm, com ele fazemos o download da dependência e depois fazemos o link utilizando as tags link e script

Sintaxe do Bootstrap

Em geral utilizamos as classes do Bootstrap para adicionar funcionalidades ao elemento atráves do atributo class

Breakpoints do Bootstrap

Breakpoints são pontos especificos de resolução de tela utilizados para deixar a aplicação responsiva e o Bootstrap tem 6 Breakpoints que são:

XS - Extra small

A propriedade é aplicada se o dispoditivo tiver menos que 576px

SM - small

A propriedade é aplicada se o dispoditivo tiver 576px a 767px

MD - medium

A propriedade é aplicada se o dispoditivo tiver 768px a 991px

LG - large

A propriedade é aplicada se o dispoditivo tiver 992px a 1199px

XL - extra large

A propriedade é aplicada se o dispoditivo tiver 1200px a 1399px

XXL - extra extra large

A propriedade é aplicada se o dispoditivo tiver 1400px ou mais

Grid

O sistema de grid do bootstrap é dividido em 12 colunas e 6 breakpoints padrões o que cobre a maioria dos dispositivos e é feito utilizando:

Containers

Container são muito importantes para estruturar o layout em areas, container são parte do grid e tem 3 formas de usar que são:

Container

Container é uma classe que cria um container responsivo com uma largura fixa que muda de acordo com a resolução da tela.

Container-breakpoints

Também podemos utilizar os Breakpoints do Bootstrap se queremos que ele funcione apenas em algum tipo de resolução ao invés de todos ex:
container-sm

Container-fluid

Container-fluid utiliza 100% da largura útil em qualquer resolução de tela

Bootstrap icons

O que é o Bootstrap icons

Bootstrap icons é a biblioteca de icones do bootstrap, ela é open source e pode ser usada tanto com o Bootstrap quanto sem ele

Qual o objetivo do Bootstrap icons

O objetivo do Bootstrap icons é fornecer uma biblioteca de icones gratuita para não precisarmos usar icones de terceiros

Como inserir o Bootstrap no projeto

Bootstrap icons tem 3 formas mais usadas de ser inserido no projeto que são:

Utilizando um CDN

Podemos inserir a biblioteca no nosso projeto utilizando um CDN

Baixar os icones

Podemos baixar os icones direto do site e utilizarmos conforme necessario

Utilizando gerenciadores de pacotes

Podemos utilizar os gerenciadores de pacotes para baixar a biblioteca

Como utilizar

Podemos utilizar os icones de diversas formas, porém as mais usadas são

Incorporar

Podemos utilizar o icone que quisermos apenas incorporando ele ou seja copiando o código svg do site do Bootstrap icons e colando no nosso projeto

Como imagem externa

Usamos ele utilizando a tag img + o atributo source igual qualquer imagem

Como uma fonte

Usamos ele utilizando a tag i + a classe bi + a classe do icone que queremos, com isso podemos alterar ele como se faz em fontes
